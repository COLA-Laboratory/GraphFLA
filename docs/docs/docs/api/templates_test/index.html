    def graphfla.analysis.classify_epistasis(landscape, approximate=False, sample_cut_prob=0.2) → Dict[str, float]

    Calculates the prevalence of five types of epistasis in a fitness landscape. Epistatic interactions are classified into positive, negative, magnitude, sign, and reciprocal sign epistasis based on pairwise mutation analysis using the motif function in igraph.

    Parameters

    Name Type Default Description
    landscape Landscape required The fitness landscape object.
    approximate bool False If True, estimates motif counts instead of exact enumeration. Recommended for large landscapes (>10,000 mutants).
    sample_cut_prob float 0.2 Pruning probability for the search tree. Only used when approximate=True. Higher = faster but less accurate. Must be in [0, 1].

    Returns

    Type Description
    Dict[str, float] Dictionary mapping 5 epistasis types to their proportions. Positive + negative sum to 1; magnitude + sign + reciprocal sign sum to 1.

    Warning

    Computationally expensive for large landscapes (>10,000 mutants). Use approximate=True with a higher sample_cut_prob value.

    >>> classify_epistasis(landscape, approximate=True, sample_cut_prob=0.1)
{'magnitude epistasis': 0.909, 'sign epistasis': 0.076,
 'reciprocal sign epistasis': 0.016, 'positive epistasis': 0.365,
 'negative epistasis': 0.635}

    def graphfla.analysis.classify_epistasis(landscape, approximate=False, sample_cut_prob=0.2) → Dict[str, float]

    Calculates the prevalence of five types of epistasis in a fitness landscape.

    Epistatic interactions are classified into five types based on pairwise mutation analysis: positive, negative, magnitude, sign, and reciprocal sign epistasis. Uses the motif function in igraph internally.

    Note

    The fractions for positive and negative epistasis always sum to 1 (mutually exclusive and exhaustive). Likewise for magnitude, sign, and reciprocal sign.

    Warning

    Computationally expensive for large landscapes (>10,000 mutants). Use approximate=True for better performance.

    landscape : Landscape
    The fitness landscape object.
    approximate : bool, default=False
    If True, estimates motif counts instead of exact enumeration. Recommended for large landscapes (>10,000 mutants).
    sample_cut_prob : float, default=0.2
    Pruning probability for the search tree. Only used when approximate=True. Higher values lead to faster computation but potentially less accurate estimations. Must be between 0 and 1.
    Dict[str, float]
    A dictionary where keys are the 5 epistasis type names and values are their calculated proportions.
    Key Range
    'magnitude epistasis' [0, 1]
    'sign epistasis' [0, 1]
    'reciprocal sign epistasis' [0, 1]
    'positive epistasis' [0, 1]
    'negative epistasis' [0, 1] 
    from graphfla.analysis import classify_epistasis

result = classify_epistasis(landscape, approximate=True, sample_cut_prob=0.1)
print(result)
# {'magnitude epistasis': 0.909, 'sign epistasis': 0.076,
#  'reciprocal sign epistasis': 0.016, 'positive epistasis': 0.365,
#  'negative epistasis': 0.635}

    C1

    class graphfla.landscape.Landscape(type='default', maximize=True)

    Fitness landscape represented as a directed graph where nodes are genotypes and edges connect mutational neighbors pointing towards fitter variants.

    Constructor Parameters

    Name Type Default Description
    type str 'default' Landscape type. One of 'default', 'boolean', 'dna', 'rna', 'protein'.
    maximize bool True If True, higher fitness is better (peaks are optima). If False, seeks lower values.

    Attributes

    Name Type Description
    graph igraph.Graph The directed graph representing the fitness landscape.
    n_configs int Total number of configurations (nodes) in the landscape.
    n_vars int Number of variables defining a configuration.
    n_lo int Number of local optima in the landscape.
    go int Node index of the global optimum.

    Methods

    .build_from_data(X, f, data_types=None, calculate_basins=True, verbose=True) → None

    Construct the landscape graph from configuration data. Populates core attributes and enables analysis.

    Name Type Default Description
    X DataFrame required Configuration data where rows are genotypes and columns are variables.
    f Series required Fitness values for each configuration. Must match length of X.
    data_types dict None Dictionary specifying variable types ('boolean', 'categorical', 'ordinal').
    calculate_basins bool True Whether to compute basins of attraction for each local optimum.
    verbose bool True Print progress during construction.

    .get_data(lo_only=False) → pandas.DataFrame

    Extracts landscape data as a pandas DataFrame where rows are configurations and columns are attributes.

    Name Type Default Description
    lo_only bool False If True, returns only local optima data (with LON attributes if available).

    .describe() → None

    Prints a summary of the landscape including size, optima count, and analysis status.

    Example

    from graphfla.landscape import Landscape

landscape = Landscape(type="boolean", maximize=True)
landscape.build_from_data(X, fitness, verbose=False)
print(f"Configurations: {landscape.n_configs}")
print(f"Local optima: {landscape.n_lo}")
landscape.describe()

    C5

    class graphfla.landscape.Landscape(type='default', maximize=True)

    Fitness landscape represented as a directed graph where nodes are genotypes and edges connect mutational neighbors pointing towards fitter variants. Landscape can be constructed from raw data (build_from_data) or from an existing graph (build_from_graph).

    Constructor

    type : str, default='default'
    Landscape type. One of 'default', 'boolean', 'dna', 'rna', 'protein', 'sequence'.
    maximize : bool, default=True
    Optimization direction. True = maximize fitness (peaks are optima).

    Attributes

    graph : igraph.Graph
    The directed graph representing the fitness landscape.
    n_configs : int
    Total number of configurations (nodes) in the landscape.
    n_vars : int
    Number of variables defining a configuration.
    n_lo : int
    Number of local optima in the landscape.
    go : int
    Node index of the global optimum.
    shape : tuple
    (n_configs, n_edges) of the landscape graph.

    Methods

    .build_from_data(X, f, data_types=None, calculate_basins=True, verbose=True) → None

    Construct the landscape graph from configuration data.

    X : DataFrame
    Configuration data (rows = genotypes, columns = variables).
    f : Series
    Fitness values for each configuration.
    data_types : dict, optional
    Variable type specification. Required when type='default'.
    calculate_basins : bool, default=True
    Whether to compute basins of attraction for each local optimum.
    verbose : bool, default=True
    Print progress during construction.

    .get_data(lo_only=False) → pandas.DataFrame

    Extract landscape data as a pandas DataFrame.

    lo_only : bool, default=False
    If True, returns only local optima data with LON attributes when available.

    .describe() → None

    Print a summary of the landscape properties, including size, optima count, and analysis status.

    Example

    from graphfla.landscape import Landscape
import pandas as pd

X = pd.DataFrame({'var_0': [0, 0, 1, 1], 'var_1': [0, 1, 0, 1]})
f = pd.Series([1.0, 2.0, 3.0, 2.5])

landscape = Landscape(type="boolean", maximize=True)
landscape.build_from_data(X, f, verbose=False)

print(landscape)
# Landscape with 2 variables, 4 configurations, ...
