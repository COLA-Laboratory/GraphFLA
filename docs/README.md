# EMOC+X: Evolutionary Multi-objective Optimization in C++ for Real World Applications
Welcome to the documentation for EMOC+X (**E**volutionary **M**ulti-objective **O**ptimization in **C**++ for Real World Applications) which is a framework for single-objective and multi-objective optimization. This project uses MkDocs to generate its documentation site. MkDocs is a fast, simple and downright gorgeous static site generator that's geared towards building project documentation. Documentation source files are written in Markdown, and configured with a single YAML configuration file.

## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites
- Python 3.5 or later
- Pip (Python package manager)

### Installing MkDocs
To install MkDocs, run the following command in your terminal:
```shell
pip install mkdocs-meterial
```

### Building the Project
When you're ready to build a static site from your documentation, run:
```shell
mkdocs build
```

This command builds your documentation into static HTML files located in the `site` directory, ready to be deployed.

### Serving the Project Locally
To serve your project locally and view it in your browser, run:
```shell
mkdocs serve
```

This command starts a local server. Open your browser and visit `http://127.0.0.1:8000/` to view your project's documentation.
